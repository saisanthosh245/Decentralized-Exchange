# Decentralised Exchange

## Project Description
The *Decentralised Exchange* is a smart contract-based prototype that allows users to add liquidity and perform token swaps between different token pairs. It simulates the core functionality of an automated market maker (AMM) with a simple pricing mechanism.

## Project Vision
To build a decentralized, trustless, and permissionless platform for token exchange that empowers users to trade without relying on centralized intermediaries.
